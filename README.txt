Para usar nossa aplicação de transmissão de arquivos de forma confiável, utilizando o protocolo rdp 3.0, os passos são:

1. Inicie primeiro o receiver usando o comando: python3 ./receiver.py
2. Após iniciar o receiver, bote o arquivo que você deseja transferir no mesmo diretório do transmitter
3. Inicie agora o transmitter com o comando: python3 ./transmitter.py (NOME_DO_SEU_ARQUIVO)
